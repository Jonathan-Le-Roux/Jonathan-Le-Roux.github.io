AI Impact on Students' Success

This project analyzes how AI tools affect student academic performance using a dataset of 400 students. It explores whether factors like AI tool choice, daily usage time, purpose, age, and satisfaction level are related to grade outcomes.

Python Pandas & NumPy Ecoders and Scalers Scikit-learn (Logistic Regression, Random Forest) Matplotlib / Seaborn
More details

Problem

As students increasingly use tools like ChatGPT, Gemini, Copilot, Grammarly, and Notion AI, it is important to understand whether AI helps or hurts academic performance. The project investigates whether AI usage patterns can explain or predict whether a student’s grades improve, slightly decline, or stay the same.

Approach

I performed exploratory data analysis to understand student demographics, AI tool usage, daily usage hours, purposes for using AI, and reported grade impact. Then I preprocessed the data by encoding categorical variables, scaling numeric features, splitting the data into training and testing sets, and testing classification models including logistic regression and a random forest classifier.

Results & Impact

The logistic regression model achieved about 30% accuracy, which suggests the available features were not strong predictors of grade impact. The random forest model showed that daily usage hours, age, and satisfaction level were the most important features, but their importance scores were still relatively low, indicating that the dataset may not contain enough information to reliably predict how AI affects student grades.

Movie Recommendation System

This project builds a personalized movie recommender system using the MovieLens ml-32m dataset. It uses movie ratings, genres, release year, and Bayesian averages to recommend similar movies with a KNN-based model.

Python Pandas & NumPy Scikit-learn (KNN) Matplotlib / Seaborn MovieLens Dataset
More details

Problem

With thousands of movies available, users need a way to discover films that match their preferences. The challenge is to create recommendations that are useful, personalized, and based on meaningful similarity rather than just popularity or average rating.

Approach

I explored and cleaned the MovieLens data, analyzed rating patterns, genre distributions, and popularity trends, then created Bayesian average ratings to reduce the impact of movies with very few ratings. I transformed movie genres into numeric features, scaled the data, and used a KNN model to find the five most similar movies to a given title.

Results & Impact

The model was able to recommend movies with similar genre and rating patterns, such as recommending animated adventure films for Toy Story. However, some recommendations had weak genre overlap, showing that future improvements could include features like actors, directors, plot descriptions, or user demographics to improve recommendation quality.

NYC 311 Complaint Resolution Time Prediction with AWS

This project uses AWS cloud tools to analyze and model New York City 311 service request data. It focuses on predicting complaint resolution time based on request characteristics using scalable data processing and machine learning techniques.

Python AWS S3 AWS Athena (SQL) AWS Sagemaker AI scikit-learn
More details

Problem

City governments collect massive amounts of service request data, but extracting meaningful insights from it can be difficult due to its size and complexity. The goal of this project is to efficiently process and analyze NYC 311 data to predict how long it would take for a complaint to be resolved based on various features.

Approach

I stored and queried the NYC 311 dataset using AWS services such as S3 and Athena to handle large-scale data efficiently. After cleaning and transforming the data, I engineered features such as complaint type, location, and time-based variables, then built and evaluated predictive models to estimate resolution time.

Results & Impact

The models were able to capture patterns between complaint characteristics and resolution time, showing that factors like complaint type, agency and location significantly influence how quickly issues are resolved. While predictions were not perfectly precise, the results demonstrate that cloud-based data pipelines combined with machine learning can provide useful estimates to support operational planning and resource allocation.
